# gRPC playground
## Description

Simple API that can request some methods in gRPC

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Invoke
 - Use Postman
 - Load proto
 - Select some method and invoke on localhost:5000
